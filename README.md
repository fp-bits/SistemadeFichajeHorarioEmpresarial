#  Desarrollo e implementación de un sistema de fichaje horario empresarial

• Descripción: El control horario de los trabajadores es una necesidad fundamental en las
empresas modernas, tanto desde el punto de vista organizativo como legal. La digitalización
de este proceso permite mejorar la fiabilidad de los datos, reducir errores humanos y facilitar
la gestión administrativa.
• Objetivo general: Automatizar el registro de la jornada laboral cumpliendo la normativa y
facilitando informes al departamento de RRHH.
• Objetivo especifico:
- Registrar entradas y salidas de trabajadores.
- Generar informes mensuales de horas trabajadas.
- Permitir fichar desde varios dispositivos (PC, móvil, lector RFID…).
